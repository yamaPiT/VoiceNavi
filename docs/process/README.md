# docs/process/ ディレクトリについて

本ディレクトリは、DADAプロセスの**状態管理と文脈復元**を担うファイルを格納します。

## 格納ファイル

| ファイル | 役割 | 更新タイミング |
| :--- | :--- | :--- |
| `last_phase_summary.md` | フェーズ移行時のサマリー | `context-reset` スキル実行時にAIが自動更新 |
| `phase5_evaluation_guide.md` | Phase 5 評価ガイド（推奨） | テンプレート。必要に応じてProduct Ownerが参照 |

## 運用ルール

1. **AIエージェント**は、フェーズの承認ゲート通過後に `context-reset` スキルを実行し、`last_phase_summary.md` に現在の状態を書き出します。
2. **新しいチャット**を開始した際、AIエージェントはまず本ディレクトリの `last_phase_summary.md` を読み込み、前回の文脈を復元してから作業を開始します。
3. 本ディレクトリのファイルは**人間が直接編集する必要はありません**。ただし、内容を確認して状態を把握することは推奨されます。
