# 外部調査スキル (Context7 MCP連携)

本スキルは、ローカル環境（PC内部）に存在しない最新の技術ドキュメントやAPI仕様を、外部から正確に取得するための手順書です。

## 実行手順 (Mechanical How)

1. **発動条件**:
   - コードの生成、環境のセットアップ、設定手順の確認時に「最新の公式情報が必要だ」とRoleが判断した場合、自律的に本スキルを実行してください。
2. **調査の実行**:
   - context7 MCPサーバーを利用し、指定された技術の最新公式ドキュメントを検索・取得します。
3. **推測の排除**:
   - 取得した一次情報（事実）のみを抽出し、勝手な推測や古い学習知識を混ぜずに、思考を担当しているRole（アーキテクトやプログラマー）へ情報を渡してください。
